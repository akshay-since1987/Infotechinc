Declaration

In Many languages you declare variables like int x = 10;, but in Javascript it always has to be var x = 10; and Javascript will always recognize the type of the variable inherently.

Examples of variables

  • var x = undefined;
  • var x = 10;
  • var x = 10.2;
  • var x = "Some String";
  • var x = "C";
  • var x = 'Some String again';
  • var x = 'C';
  • var x = 10e2;
  • var x = ["a", "b", 3, 4, "e", "6","10e3"];
  • var x = {a: "B"}

Types of Variable and Some fun

That leads us to Types. What decides the type of the Variables?

The type of the variable is always determined by the current value it holds.

Examples: 

				var x;
				console.log(typeof x);
				var x = 0;
				console.log(typeof x);
				var x = 10;
				console.log(typeof x);
				var x = 10.2
				console.log(typeof x);
				var x = "String";
				console.log(typeof x);
				var x = "S";
				console.log(typeof x);
				var x = 'S';
				console.log(typeof x);
				var x = [1,2,3];
				console.log(typeof x);
				var x = ['a',"b", "C"];
				console.log(typeof x);
				var x = ["A", 1, 2];
				console.log(typeof x);
				var x = {a: "b"};
				console.log(typeof x);
